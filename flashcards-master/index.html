JavaScript

JavaScript is a lightweight interpreted or just-in-time compiled programming language.

Operator

Operator performs some operation on single or multiple operands (data value) and produces a result.

Variable

Variable is a container for storing data values.

Variable naming convention

Variable naming convention for JavaScript are the following: names should not start with a numeral (0-9). They must begin with a letter or an underscore character. They are are case-sensitive.

Function

Function is a block of code designed to perform a particular task..

Method

Method is an action that can be performed on objects.

Argument

Argument is a value passed to the function when it is invoked.

Parameter

Parameter is a variable listed as a part of the function definition.

Return

Return statement stops the execution of a function and returns a value from that function.
